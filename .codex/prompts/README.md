# Codex Prompt Templates

These are reusable prompt starters for common Codex tasks.

Codex does not auto-register them as slash commands. Copy the template you want, fill in the placeholders, and paste it into your Codex task.

Most prompts here assume:

- `AGENTS.md` is the base instruction file
- `CONTEXT.md` holds live project state
- `.codex/agents/` contains optional role briefs

Start with the smallest prompt that matches the task.
